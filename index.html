台灣六縣市景點介紹

本網頁內容主要分享介紹台灣六個縣市的景點，分別為新北、宜蘭、花蓮、台東、台南、澎湖
從各縣市中挑選出兩個推薦的景點做介紹、分享!

新北(New Taipei City)

三貂角燈塔步道

1.地址：新北市貢寮區福連村馬崗街38號。
2.電話：02-24991300。
3.燈塔開放時間：09:00~18:00(夏令時間4月1日至10月31日)，0900~17:00(冬令時間11月1日至3月31日)；休週一。
<三貂角燈塔步道交通/怎麼去>
1.開車：Google Map可用「三貂角燈塔」導航。
2.公車：搭乘國光客運1740至「啟源站」；國光客運1811或1812至「啟源站」；新巴士F831至「萊萊站」。 

Image Description

<介紹>

三貂角燈塔是全台灣最早看到第一道曙光的地方，也是第一座對外開放參觀的燈塔。西元1935年完工，至今已有近100年歷史，更是台灣第一座免費開放參觀的燈塔，在各方面都有著極為重要的地位與意義。蔚藍美麗的東北角海岸景色，加上類似歐式教堂風格的建築是很多人都會來拍照打卡的台北IG景點，目前內部尚未開放，外部可供遊客取景拍照，推薦大家不妨有空去走走，運動之外，還可以欣賞美麗海景。

觀音山林梢步道

1.地址：新北市五股區凌雲路三段130號(觀音山遊客中心)。
2.電話：02-22928888(觀音山遊客中心)。
3.開放時間：24H。
<林梢步道交通/怎麼去>
1.開車：Google Map可用「觀音山遊客中心」導航，步道入口就在遊客中心下方的生態園區。
2.公車：搭乘捷運蘆洲線至蘆洲站，於3號出口搭橘20至觀音山遊客中心；或搭乘785（經捷運北門站）至凌雲寺，由此走林梢步道第二段往遊客中心接第一段。

Image Description

<介紹>

林梢步道是一條建構於林間的天空步道，景緻優美，分成兩段式的步道加起來距離只有800公尺，來回只需30分鐘，非常適合親子旅遊、銀髮族健行的景點，步道平緩好走，沿途多有樹蔭遮蔽，觀景台還能遠眺淡水河、台北市景，CP值超高。可接續前往周圍許多景點一同遊玩，也可以搭渡輪去關渡淡水走走。

宜蘭(Yilan City)

宜蘭外澳飛行傘基地

1.地址：宜蘭縣頭城鎮石空路95號
2.營業時間：09:00 ~ 15:00 !注意：會視氣候開放!
3.大眾運輸(開車路線)
台北→宜蘭：國道五號→下頭城交流道→右轉台2庚線→台2線→左轉接濱海路二段一巷→石空路→「外澳-伯朗咖啡城堡」。
4.火車：搭乘火車→於外澳火車站下車→轉搭計程車→「外澳-伯朗咖啡城堡」。
5.體驗費用:
外澳:體驗飛行高度160公尺，約飛3~10分鐘，單次1200元/人
北宜61K：體驗飛行高度350公尺，約飛3~20分鐘，單次1500/人 

<介紹>

許多人都想嘗試看看跳傘、飛行傘這種比較高難度的活動，但台灣目前可以體驗飛行傘的地方並不多，且在宜蘭外澳體驗飛行傘，腳下是太平洋，還能將蘭陽平原和龜山島一覽無遺，非常推薦想體驗此種刺激活動的遊客來嘗試。

國立傳統藝術中心

1.園區地址:宜蘭縣五結鄉五濱路二段201號
2..營業時間：每日9：00〜18：00
3.票價:普通票150/人，一般遊客
<交通資訊>
4.自行開車 
由國道五號高速公路，於羅東（五結）交流道下，前行至傳藝路，由迴轉道左轉傳藝路，經傳藝大橋至北部濱海公路（台二省道），左轉後即可抵達。
5.火車: 臺北或花蓮出發，至羅東火車站下車。
6.客運(國光客運):1791 羅東－南方澳(經清水)，至大眾廟-傳藝中心下車，步行即抵達園區。

<介紹>

傳統藝術中心是一個非常適合親子的旅遊景點，園區內有許多DIY體驗的活動，也有很多傳統的民俗技藝，很多都是獨特的中華文化，不管是和孩子一起體驗了解，或是自己和家人朋友去走走看看觀賞，都是一個很棒的體驗，也是一個可以好好了解中華文化的機會，推薦大家去遊玩。

花蓮(Hualien County)

親不知子天空步道

1.地點：親不知子天空步道 (豐濱天空步道)
2.地址：花蓮縣豐濱鄉台11線新峯21之2號
3.營業時間：09:00–17:00 (周一至周日)
<專屬停車場收費>
遊覽車150元、汽車50元、機車20元 

<介紹>

親不知子天空步道，早年是穿越親不知子斷崖的原住民步道，與廢棄的新磯隧道平行。不知道是什麼原因，在1971年就閒置荒廢了，而近期在花蓮縣豐濱鄉公所的努力下，重新打造成全新的天空步道，成為當前花蓮最夯景點之一。整個步道分成二段，一段是原有古道透過水泥加固、拓寬；另一段垂直懸空的懸崖，則以鋼骨跟強化玻璃興建，沿途中盡享無敵海景，來到花蓮一定要來嘗試走這個步道。

瑞穗牧場

1.地點：瑞穗牧場
2.地址：花蓮縣瑞穗鄉6鄰157號
3.開放時間：09:00-18:00 (周一至周五)；08:00-17:00 (周六日)
4.門票：免費
<瑞穗牧場交通>
開車為主，牧場前有整排路邊免費停車場，非常方便

<介紹>

來瑞穗牧場門票免費、停車也免費、看牛、看鴕鳥、喝濃純鮮奶，是非常受歡迎也非常推薦的花蓮景點，整個牧場腹地其實不大，最精彩的是這裡有一大片大草坪，適合親子休閒、玩樂。瑞穗也有自己生產的美食，像鮮乳捲、鮮奶、鮮乳酪！尤其超人氣鮮乳非常受歡迎～周邊住宿也頗多。附近景點推薦接續北回歸線、吉蒸牧場一同遊玩，此外臨行前還可以去逛逛伴手禮店，牧場內牛軋糖也很有名。

台東(Taitung County)

金剛大道

1.地址：臺東縣長濱鄉東13鄉道(長光產業道路)
2.開放時間:24HR(周一至周日) 
3.電話：089-832139

<介紹>

位在台11線約85公里處的東13鄉道，是忠勇自行車道的一部分。其中有一段道路，兩旁沒有電線杆，筆直且依山傍海的絕妙風景，被比喻為東海岸的伯朗道。在這段道路上可以遠眺西側海岸山脈山勢俊朗的「金剛山」，因而被稱「金剛大道」，搭配道路兩旁的梯田景觀，風景隨季節變化，令人百看不膩。推薦大家可以來踏踏青，欣賞廣大美麗的農村風景。

星星部落觀景咖啡廳

1.地址：台東縣卑南鄉富源村富源 116-3 號
2.電話：08-9238089
3.營業時間：18:00-01:30 (週日休)
4.價錢:入園賞夜每人低消為$50／人

<介紹>

台東也有超美的夜景景觀餐廳！距離台東市區不到 30 分鐘的路程，位在卑南鄉的星星聚落景觀咖啡是來到台東絕對不能錯過的夜景景點，除了賞夜景外，星星部落餐廳提供豐富的飲品選擇，有咖啡、茶飲、果汁和冰沙等(左圖)，另外也有炸物輕食和現泡泡麵，適合三五好友或情侶來這裡閒聊、放鬆，推薦愛看夜景的人來逛逛。

台南(Tainan City)

楠西梅嶺

1.地址：715台南市楠西區香蕉山29-2號
2.連絡電話：06-6900399
3.開放時間:24HR(每日)
4.門票資訊：無門票費用
<交通資訊(開車)>
國道一號：由麻豆交流道下，循171縣道轉台1線，過渡頭橋左接台84線至玉井，左轉台3線過楠西橋即可到達梅嶺風景區。
國道三號：由新化交流道下，循台20線往玉井、楠西方向右轉南188鄉道即可到達梅嶺風景區。

<介紹>

梅嶺位於楠西區灣丘里，為開放山區，其中包含了多條不同等級的步道。舊時以種植香蕉為主，固有香蕉山之舊稱，後才改種梅樹，名稱也就成了梅嶺。在5、6月份到梅嶺能夠三重享受一次滿足，桐花鋪滿一地，賞一場五月雪；梅花的白色花季過去，農人們開始採收翠綠梅子，當地的特色梅餐絕對能滿足你的味蕾；入夜後再走入山林，與火金姑來一場夏日約會。

七股鹽山

1.地址：台南市七股區鹽埕里66號
2.營業時間:夏季:3月-10月9 :00~18:00/冬季:11月-2月8:30~17:30
3.門票：單人50元、全車100元
<交通資訊(開車)>
國道1北上，下麻豆交流道往佳里區，接176縣道往七股，沿途請依「七股鹽山」路標行駛即可抵達。
國道1南下，下營系統交流道接84線快速道路往西，至北門交流道接濱海61線快速道路往南，下七股鹽埕交流道後左轉176線道，直走約1500公尺即可抵達。

<介紹>

七股鹽山所在地是過去的七股鹽場，曾是全台最大的曬鹽場，因曬鹽業式微2002年停止營業，轉型為觀光園區。園區很大，除了最經典的鹽山，還有不少裝置藝術和遊樂設施，個人覺得最值得的，就是在園區買一支加了鹽霜淇淋，非常好吃，鹹甜鹹甜的口味，也是只有園區才有，此外販賣部也販售不少獨特的產品，其他園區美食也值得品嘗，推薦大家去玩~

澎湖(Penghu County)

海洋皇宮海洋世界

1.地址：澎湖縣馬公市臨海路36-1號3F
2.搭船地點：南海遊客中心-海上皇宮櫃台報到
3.船班時間：08:30、9:00、10:00、11:30、
13:00、14:00、15:30、16:30、17:00
4.價錢(行程活動價格)
【海洋牧場】 成人(12歲以上)：600元 兒童(3~12歲)：400元 3歲以下：保險費 50
【夜釣小管】 統一價：500元 3歲以下：保險費50
【花火專船】 統一價：350元 3歲以下：保險費50

<介紹>

來到澎湖，可以在搭船之餘，看看海底世界，也可以品嘗許多澎湖美食。海洋皇宮為滿天星航運公司旗下澎湖在地旅遊產業，目前為澎湖最大最新穎的海上休閒活動平台，活動空間安全、舒適、寬敞，同時間可容納多數旅客之海洋休閒平台，平台上除了碳烤牡蠣，海產粥、釣海鱺魚、溜花枝、和餵食其他魚群外亦提供其他澎湖美食小吃，在船上吃美食，是個特別的體驗，推薦大家來體驗一番。

海底郵筒浮潛體驗

1.活動時長:2 小時
2.活動場次:上午場:10:00-12:00/下午場:13:00-15:00/15:30-17:30(加開場，非每日有)
3.活動流程：
1.波賽頓集合報到
2.填寫資料與切結書，如需投遞明信片可另外加購($100/張)
3.更換下水衣物及領取助浮衣、浮潛裝備（面鏡、呼吸管、潛水靴）
4.陸上教學（裝備穿著、面鏡及呼吸管使用、水上活動安全教育）
5.潛點移動（需自備交通工具移動至潛點：鎖港杭灣海底郵筒）
6.下水體驗（水上複習、浮潛體驗、郵筒投遞{需有自潛證照}）
7.上岸回到店內盥洗 

<介紹>

澎湖海底郵筒是全台第一個設置於水底的郵筒，結合了裝置藝術與珊瑚礁復育的功能，目前也成為澎湖熱門景點之一 ，親自潛水到海底與澎湖藝術魚礁海底郵筒見面，並且寄出愛的明信片！
